The THEMENAME theme is a customizable new theme for WordPress.  THEMENAME integrates many WordPress features including feature images for posts, customizable menus, and widgetized sidebars and footer!

THEMENAME also has fully integrated social sharing links, advanced search engine optimization features including post, page, and category level settings, and more!

THEMENAME also supports the very popular Yet Another Related Posts Plugin.  Just activate the plugin and popular posts will be displayed below each post with the post thumbnail!
